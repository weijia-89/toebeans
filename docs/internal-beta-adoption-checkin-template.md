# Internal beta: adoption check-ins (template)

Use this for written tester updates during M1.2. Copy/paste into a text message, email, or a shared doc.

## Daily (2 minutes)

- Date:
- Did you get any reminder notifications today? (Y/N)
- Did you log any doses today? (Y/N)
- Any missed or late doses because of the app? (Y/N, short note)
- Anything confusing or annoying? (one sentence)

## Weekly (10 minutes)

- Week ending:
- How many days this week did you use the app? (0–7)
- Any day you stopped trusting reminders? What happened?
- Most helpful thing the app did this week:
- Biggest friction (tap count, wording, navigation, timing, etc.):
- If you could change one thing, what would it be?

## If something went wrong (bug report)

- What did you expect to happen?
- What happened instead?
- Approx time:
- Phone model + Android version:
- Screenshot (if possible):
- If the app crashed: Settings → Diagnostics → Export crash log
